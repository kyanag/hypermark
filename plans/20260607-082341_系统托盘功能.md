# 系统托盘功能

## 需求概述

将 HyperMark 从命令行黑框框应用升级为带有系统托盘的后台服务应用，提供以下功能：

1. **系统托盘图标** - 应用启动后在 Windows 系统托盘区域显示图标
2. **服务开关** - 通过托盘右键菜单可以关闭/开启 HTTP 服务
3. **开机启动设置** - 通过托盘菜单可以设置是否开机自动启动

## 当前状态

- 项目类型：ASP.NET Core Web API 服务（.NET 10.0）
- 启动方式：`dotnet run` 或发布后的 `HyperMark.Web.exe`
- UI 组件：无（纯命令行）
- 发布目标：win-x64 self-contained，已启用 AOT

## 功能详细说明

### 1. 系统托盘图标

**行为描述：**
- 应用启动后，在 Windows 系统托盘（通知区域）显示 HyperMark 图标
- 双击托盘图标可以打开/隐藏控制窗口（如果有控制窗口的话）
- 托盘图标显示 Tooltip："HyperMark - 网页收藏服务"

**图标资源：**
- 需要准备 `.ico` 格式的图标文件
- 建议尺寸：16x16, 32x32 多尺寸
- 图标风格：简洁，与 HyperMark 品牌一致

### 2. 服务开关功能

**右键菜单项：**
```
┌─────────────────────────┐
│ ✓ 服务运行中            │  ← 状态显示（不可点击）
├─────────────────────────┤
│   暂停服务              │  ← 点击后变为"启动服务"
│   打开 Web 界面         │  ← 在浏览器中打开 http://localhost:5000
├─────────────────────────┤
│   设置                  │  ← 子菜单
│   退出                  │  ← 退出应用
└─────────────────────────┘
```

**服务状态切换：**
- 运行中 → 点击"暂停服务" → 停止 HTTP 监听，托盘图标变灰
- 已停止 → 点击"启动服务" → 重新启动 HTTP 监听，托盘图标恢复正常
- 状态切换时更新菜单项文本和图标

**技术实现要点：**
- 需要将 `WebApplication` 的生命周期与托盘菜单联动
- 使用 `IHostApplicationLifetime` 控制应用启停
- 或者使用 `WebApplication.StopAsync()` / `WebApplication.StartAsync()`

### 3. 开机启动设置

**设置菜单位置：**
```
设置 →
  ┌─────────────────────────┐
  │ ✓ 开机自动启动          │
  │   开始最小化            │  ← 可选：启动后最小化到托盘
  └─────────────────────────┘
```

**实现方式：**
- 使用 Windows 注册表：`HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run`
- 键名：`HyperMark`
- 键值：应用可执行文件的完整路径

**配置持久化：**
- 在 `~/.hypermark/config.json` 中添加 `startup` 配置节：
```json
{
  "http": "http://localhost:5000",
  "startup": {
    "autoStart": false,
    "startMinimized": true
  }
}
```

## 技术方案

### 方案选择：WinForms + NotifyIcon

**推荐理由：**
1. WinForms 的 `NotifyIcon` 组件是系统托盘的标准实现
2. 与 .NET AOT 兼容性好
3. 轻量级，不需要引入 WPF 的额外开销
4. 可以与 ASP.NET Core 的 `BackgroundService` 共存

**备选方案：**
- WPF：更现代，但对于纯托盘应用过于重量级
- 第三方库（如 Hardcodet.NotifyIcon.Wpf）：增加外部依赖
- 纯 Win32 API P/Invoke：过于复杂

#### 2. 托盘图标实现

```csharp
public class TrayIcon : IDisposable
{
    private NotifyIcon _notifyIcon;
    private ContextMenuStrip _contextMenu;
    private WebApplication _app;
    private bool _isRunning = true;

    public TrayIcon(WebApplication app)
    {
        _app = app;
        InitializeTrayIcon();
    }

    private void InitializeTrayIcon()
    {
        _contextMenu = new ContextMenuStrip();

        // 状态项
        var statusItem = new ToolStripMenuItem("✓ 服务运行中") { Enabled = false };
        _contextMenu.Items.Add(statusItem);
        _contextMenu.Items.Add(new ToolStripSeparator());

        // 开关按钮
        var toggleItem = new ToolStripMenuItem("暂停服务");
        toggleItem.Click += (s, e) => ToggleService(toggleItem, statusItem);
        _contextMenu.Items.Add(toggleItem);

        // 打开 Web 界面
        var openWebItem = new ToolStripMenuItem("打开 Web 界面");
        openWebItem.Click += (s, e) => OpenWebInterface();
        _contextMenu.Items.Add(openWebItem);

        _contextMenu.Items.Add(new ToolStripSeparator());

        // 设置子菜单
        var settingsMenu = new ToolStripMenuItem("设置");
        var autoStartItem = new ToolStripMenuItem("开机自动启动");
        autoStartItem.Click += (s, e) => ToggleAutoStart(autoStartItem);
        settingsMenu.DropDownItems.Add(autoStartItem);
        _contextMenu.Items.Add(settingsMenu);

        // 退出
        var exitItem = new ToolStripMenuItem("退出");
        exitItem.Click += (s, e) => ExitApplication();
        _contextMenu.Items.Add(exitItem);

        _notifyIcon = new NotifyIcon
        {
            Icon = new Icon("Resources/app.ico"),
            Text = "HyperMark - 网页收藏服务",
            ContextMenuStrip = _contextMenu,
            Visible = true
        };
    }

    private async void ToggleService(ToolStripMenuItem toggleItem, ToolStripMenuItem statusItem)
    {
        if (_isRunning)
        {
            await _app.StopAsync();
            toggleItem.Text = "启动服务";
            statusItem.Text = "⏸ 服务已停止";
            _notifyIcon.Icon = GetGrayIcon(); // 灰色图标
        }
        else
        {
            await _app.StartAsync();
            toggleItem.Text = "暂停服务";
            statusItem.Text = "✓ 服务运行中";
            _notifyIcon.Icon = new Icon("Resources/app.ico");
        }
        _isRunning = !_isRunning;
    }

    public void Dispose()
    {
        _notifyIcon?.Dispose();
    }
}
```

#### 3. 开机启动管理

```csharp
public class StartupManager
{
    private const string AppName = "HyperMark";
    private const string RunKeyPath = @"SOFTWARE\Microsoft\Windows\CurrentVersion\Run";

    public static bool IsAutoStartEnabled()
    {
        using var key = Registry.CurrentUser.OpenSubKey(RunKeyPath, false);
        return key?.GetValue(AppName) != null;
    }

    public static void SetAutoStart(bool enable)
    {
        using var key = Registry.CurrentUser.OpenSubKey(RunKeyPath, true);
        if (enable)
        {
            var exePath = Environment.ProcessPath;
            key?.SetValue(AppName, $"\"{exePath}\" --minimized");
        }
        else
        {
            key?.DeleteValue(AppName, false);
        }
    }
}
```

## 注意事项

### AOT 兼容性
- WinForms 在 AOT 模式下有一些限制
- 需要测试 `NotifyIcon` 和 `ContextMenuStrip` 是否正常工作
- 如果 AOT 不兼容，可能需要禁用 AOT 或使用方案 B

### 图标资源
- 需要准备应用图标，建议使用 SVG 转 ICO 工具
- 图标应包含多个尺寸（16x16, 32x32, 48x48, 256x256）

### 用户体验
- 应用启动后不应显示控制台窗口（WinExe 模式）
- 退出时应确认是否退出（可选）
- 服务状态变化时应有视觉反馈

### 配置文件格式
- 需要扩展 `AppConfig.cs` 以支持新的配置项
- 保持向后兼容，新配置项使用默认值
